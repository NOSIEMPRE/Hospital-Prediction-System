# 04 – Model Deployment

**Hospital Readmission Risk Prediction** — FastAPI REST API serving

---

## Overview

This module deploys the trained XGBoost model as a REST API using FastAPI and Uvicorn. The model is loaded from a local MLflow run at startup. This is the first standalone serving module — monitoring and CI/CD are added in modules 05 and 06.

---

## Contents

```text
04-deployment/
├── train.py       # Train XGBoost pipeline, log to MLflow, write run_id.txt
├── app.py         # FastAPI service (loads model from MLflow by run_id)
├── test_api.py    # Pytest integration tests for /health and /predict
├── requirements.txt
├── run_id.txt     # Generated by train.py
└── README.md
```

---

## Setup

```bash
cd 04-deployment
pip install -r requirements.txt
```

Ensure `../data/diabetic_data.csv` exists.

---

## Usage

### 1. Train

```bash
python train.py
```

Trains the model, logs metrics and artifacts to local MLflow (`mlflow.db`), and writes the run ID to `run_id.txt`.

### 2. Start API

```bash
python app.py
```

API runs at [http://localhost:9696](http://localhost:9696).

### 3. Explore

Open [http://localhost:9696/docs](http://localhost:9696/docs) for the interactive Swagger UI.

### 4. Test

```bash
python -m pytest -q test_api.py
```

Expects `2 passed`.

---

## API Endpoints

| Endpoint | Method | Description |
| -------- | ------ | ----------- |
| `/` | GET | Welcome message |
| `/health` | GET | Service status and active `run_id` |
| `/predict` | POST | Returns `risk_score` (0–1) and `model_version` |
| `/docs` | GET | Swagger UI |

### Example request

```bash
curl -X POST http://localhost:9696/predict \
  -H "Content-Type: application/json" \
  -d '{
    "time_in_hospital": 3, "num_lab_procedures": 41,
    "num_procedures": 0, "num_medications": 8,

    "change": "Ch", "diabetesMed": "Yes",
    "medication_changed": 1,
    "A1Cresult": "not_tested", "max_glu_serum": "not_tested"
  }'
```

## Troubleshooting

| Issue | Fix |
| ----- | --- |
| `run_id.txt not found` | Run `python train.py` first |
| `diabetic_data.csv not found` | Download from UCI, place in `../data/` |
| API not responding | Ensure `python app.py` is running |
